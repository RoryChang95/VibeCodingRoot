三、核心 Prompt

Prompt 1：產生 HTML 與 CSS

我是一位完全沒有程式基礎的初學者，目前正在使用 CodePen。 請幫我製作一個網頁小作品，主題是「Vibe Check！今日運勢盲盒機」。 需求如下： 1. 使用 HTML、CSS、JavaScript。 2. 這一步只需要完成 HTML 和 CSS，不需要 JavaScript。 3. 畫面要包含： - 標題：「Vibe Check！今日運勢」 - 一段文字：「點擊下方按鈕查看今日 Vibe」 - 一個按鈕：「查看今日運勢」 4. CSS 請做出漂亮的漸層背景、置中的卡片、圓角按鈕。 5. 請分別提供 HTML 和 CSS 程式碼。 6. 請在程式碼中加入適合初學者理解的中文註解。

Prompt 2：加入 JavaScript

很好，現在請幫我加入 JavaScript 互動功能。 需求如下： 1. 當使用者點擊按鈕時，隨機顯示一個今日運勢。 2. 運勢內容共有五個： - 程式碼一發就過 - 發現神級 Bug，但也順利修掉 - 咖啡差點打翻在鍵盤上 - 今日大吉，適合寫 Code - 突然看懂複雜語法 3. 每次點擊按鈕時，文字框內容都要更新。 4. 請提供 JavaScript 程式碼。 5. 請用中文註解說明每一段程式在做什麼。 6. 請確保程式可以直接貼到 CodePen 的 JavaScript 區塊執行。

四、備份程式碼

HTML 備份程式碼 
<div class="card">
  <h1>Vibe Check！今日運勢</h1>
  <p id="fortuneText">點擊下方按鈕查看今日 Vibe</p>
  <button id="fortuneBtn">查看今日運勢</button>
</div>

JavaScript 備份程式碼 
const fortuneText = document.getElementById("fortuneText");
const fortuneBtn = document.getElementById("fortuneBtn");

const fortunes = [
  "程式碼一發就過",
  "發現神級 Bug，但也順利修掉",
  "咖啡差點打翻在鍵盤上",
  "今日大吉，適合寫 Code",
  "突然看懂複雜語法"
];

fortuneBtn.addEventListener("click", function () {
  const randomIndex = Math.floor(Math.random() * fortunes.length);
  fortuneText.textContent = fortunes[randomIndex];
});

五、常見問題

我跟不上老師操作怎麼辦？

不用緊張，直播結束後可以觀看回放，也可以使用本頁提供的備份程式碼完成作品。

CodePen 一定要註冊嗎？

短時間體驗可以先不用註冊，但若想保存作品，建議註冊免費帳號。

JavaScript 貼上後沒有反應怎麼辦？

請先確認 HTML 裡面的 id 名稱是否和 JavaScript 中使用的名稱一致，例如 fortuneText 和 fortuneBtn。
